                      ADXL345 SENSOR CONNECTION
                      ═════════════════════════

Each ADXL345 sensor (5 total) connects to breadboard like this:

     ADXL345 Sensor Pin          Connects To              Example Location
     ─────────────────────────────────────────────────────────────────────
     VCC (Power)          →      Breadboard + rail        Any + hole
     GND (Ground)         →      Breadboard - rail        Any - hole
     MOSI                 →      Column B (shared)        B1, B2, B3, etc.
     MISO                 →      Column C (shared)        C1, C2, C3, etc.
     SCK                  →      Column D (shared)        D1, D2, D3, etc.
     CS (Chip Select)     →      Different column (unique)
     INT1, INT2           →      Leave disconnected       (not used)

                         ACTUAL WIRE CONNECTIONS
                         ═════════════════════════

From Arduino to Breadboard:

1. Red wire:   Arduino 5V       → Breadboard + rail (column A, + row)
2. Black wire: Arduino GND      → Breadboard - rail (column A, - row)
3. Wire 3:     Arduino pin 51   → Breadboard column B (any row 1-30)
4. Wire 4:     Arduino pin 50   → Breadboard column C (any row 1-30)
5. Wire 5:     Arduino pin 52   → Breadboard column D (any row 1-30)
6. Wire 6:     Arduino pin 53   → Breadboard column E (preferably row 2)
7. Wire 7:     Arduino pin 10   → Breadboard column F (preferably row 2)
8. Wire 8:     Arduino pin 9    → Breadboard column G (preferably row 2)
9. Wire 9:     Arduino pin 8    → Breadboard column H (preferably row 2)
10. Wire 10:   Arduino pin 7    → Breadboard column I (preferably row 2)


From Each Sensor (5 × 6 = 30 wires total):

Sensor 0:
  → VCC pin    to + rail        (any hole in + row)
  → GND pin    to - rail        (any hole in - row)
  → MOSI pin   to column B      (any hole in column B)
  → MISO pin   to column C      (any hole in column C)
  → SCK pin    to column D      (any hole in column D)
  → CS pin     to column E      (any hole in column E)

Sensor 1:
  → VCC pin    to + rail        
  → GND pin    to - rail        
  → MOSI pin   to column B      
  → MISO pin   to column C      
  → SCK pin    to column D      
  → CS pin     to column F      

Sensor 2:
  → VCC pin    to + rail        
  → GND pin    to - rail        
  → MOSI pin   to column B      
  → MISO pin   to column C      
  → SCK pin    to column D      
  → CS pin     to column G      

Sensor 3:
  → VCC pin    to + rail        
  → GND pin    to - rail        
  → MOSI pin   to column B      
  → MISO pin   to column C      
  → SCK pin    to column D      
  → CS pin     to column H      

Sensor 4:
  → VCC pin    to + rail        
  → GND pin    to - rail        
  → MOSI pin   to column B      
  → MISO pin   to column C      
  → SCK pin    to column D      
  → CS pin     to column I

================================================================================
CSV FOLDER VALIDATION SCRIPT - USAGE GUIDE
================================================================================

FILE: validate_csv_folder.py

This is a universal script that can test ANY folder containing CSV files.

================================================================================
BASIC USAGE
================================================================================

1. SIMPLE VALIDATION (just check folder for issues):

   python3 validate_csv_folder.py "/path/to/folder"

   Example:
   python3 validate_csv_folder.py "/home/itachi/data raw/datas/baseline 2082-10-3"


2. VALIDATION WITH BASELINE COMPARISON:

   python3 validate_csv_folder.py "/path/to/folder" --baseline "/path/to/baseline.csv"

   Example:
   python3 validate_csv_folder.py "/home/itachi/data raw/repaired_classified/20_good_repair" \
     --baseline "/home/itachi/data raw/datas/baseline 2082-10-3/baseline 2082-10-3 1st.csv"


3. SAVE VALIDATION REPORT:

   python3 validate_csv_folder.py "/path/to/folder" --report "report.txt"

   Example:
   python3 validate_csv_folder.py "/home/itachi/data raw/datas/damaged B1 removed" \
     --report "validation_report_damaged.txt"


4. FULL EXAMPLE (with baseline + report):

   python3 validate_csv_folder.py "/path/to/folder" \
     --baseline "/path/to/baseline.csv" \
     --report "output_report.txt"

   Example:
   python3 validate_csv_folder.py "/home/itachi/data raw/repaired_classified/20_bad_repair" \
     --baseline "/home/itachi/data raw/datas/baseline 2082-10-3/baseline 2082-10-3 1st.csv" \
     --report "bad_repair_validation.txt"

================================================================================
WHAT THE SCRIPT CHECKS
================================================================================

For each CSV file in the folder, it validates:

âœ… NaN Values
   - Checks for missing data
   - All values must be numeric

âœ… Data Types
   - All columns must be numeric (float/int)
   - Detects non-numeric data

âœ… Row Count
   - Compares with baseline if provided
   - Flags mismatches

âœ… Column Count
   - Compares with baseline if provided
   - Checks for missing/extra columns

âœ… Column Names
   - Verifies correct column structure
   - Checks for S1_X_g, S1_Y_g, S1_Z_g, S2_X_g, S2_Y_g, S2_Z_g

âœ… Extreme Values
   - Detects outliers (values > 1000 or < -1000)
   - Flags suspicious data

âœ… Quality Metrics
   - Calculates mean, std, min, max per file
   - Estimates quality score if baseline available
   - Detects degradation/recovery patterns

âœ… Statistics
   - Aggregates statistics across all files
   - Calculates average mean, std, quality scores
   - Shows quality distribution
